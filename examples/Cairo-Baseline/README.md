# Cairo Baselines

Handwritten Cairo baseline implementations used for side-by-side comparison against generated/optimized lanes.

Current baseline set:
- `newton_u128`
- `fast_power_u128`
- `fast_power_u128_p63`
- `karatsuba_u128`
- `sq128x128_u128`

Pinned provenance is tracked in `config/baselines-manifest.json`.
